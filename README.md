# Dependencies Installed
- Bootstrap 5 (alpha)
	  1) npm install bootstrap@5.0.0-alpha1
- SASS
	  1) npm install node-sass
- React Router
	  1) $ npm install --save react-router-dom
- Material-ui
	  1) npm install @material-ui/icons
	  2) npm install @material-ui/core
